# Chatbot Example

# Backend

## Installation

Python 3.6.8 is needed. Tested on Windows 10 64bit.
```
pip install -r requirements.txt
python -m spacy download en_core_web_md
```

## Train

```
doit train
doit train_core
```

## Run interactive
```
doit run_core
```

## Run Socket.io server
```
doit run_server
```

# Frontend

Run the following command in the frontend folder
```
livereload
```
